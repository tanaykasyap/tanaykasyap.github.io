
# Tanay Kasyap — Editorial Serif + Left Sidebar (Quarto)

1) Install Quarto (Mac Terminal): `brew install quarto`
2) Open this folder in VS Code.
3) Put your photos in `assets/img/`:
   - kid: `kid-tanay.jpg`
   - present: `today-tanay.jpg`
4) Preview: `quarto preview`
5) Build: `quarto render` (creates `docs/`)
6) Publish to GitHub Pages (repo name must be `tanaykasyap.github.io`, Pages → Branch: main / Folder: /docs).
